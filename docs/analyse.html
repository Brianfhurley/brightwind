Computes the distribution of first variable with respect to wind direction sectors :param var_series: Series of the variable whose distribution we need to find :param direction_series: Series of wind directions between [0-360] :param sectors: Number of sectors to bin direction to.   • direction_bin_labels – Optional, you can specify an array of labels to be used for the bins. uses string labels of the format ‘30-90’ by default
